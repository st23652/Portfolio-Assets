# 🤖💻 AI Code Commenter

**[🔗 View the Project Repository Here](https://github.com/st23652/AI_CodeCommenter)**

## About the Project
**AI Code Commenter** is a web application designed to automatically generate detailed or brief explanatory comments for raw source code using OpenAI's powerful GPT models. It bridges the gap between complex logic and readable documentation, helping developers understand, document, and maintain their codebases efficiently. 

Built with Python and Flask, the application features a responsive frontend that allows users to paste code snippets, adjust AI generation parameters, and instantly receive contextual documentation.

## Key Highlights
* **Customizable AI Generation:** Choose between generating deep, architectural explanations or brief, inline comments based on your documentation needs.
* **Model & Creativity Control:** Toggle between OpenAI models (GPT-4o or GPT-3.5-Turbo) and fine-tune the AI's creativity using a built-in temperature slider (0 to 1).
* **Developer-Friendly UI:** Features a clean, responsive interface equipped with a Dark Mode toggle for comfortable viewing day or night.
* **Instant Export Options:** Seamlessly copy the generated explanations to your clipboard or download them directly as a `.txt` file.
* **Safe Processing:** Restricts code input size to 6,000 words to ensure optimal performance and prevent API overloading.

## Technologies Used
* **Backend:** Python 3.13+, Flask
* **AI Integration:** OpenAI Python SDK
* **Frontend:** HTML5, CSS3, JavaScript
* **Environment:** `python-dotenv` for secure API key management
