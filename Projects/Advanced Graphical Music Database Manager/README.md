# Music Catalog Manager 🎵🎶

## Overview
This project is a Java-based application for organizing and visualizing a music album collection. It uses a structured object-oriented design to model individual albums and their details, a robust database layer for collection management, and a custom-designed graphical component to display the data dynamically in an interactive GUI.

## Key Features
* **Structured Album Model:** Encapsulates detailed metadata (title, artist, year, tracks) for each album.
* **Database Collection Layer:** Handles album collection logic, including loading, storage, and retrieval operations.
* **Custom Visual Component:** Implements a unique graphical rendering approach for presenting album information clearly.
* **Integrated Main Application:** Cohesively links data models, database management, and the visual display within a cohesive main application window.

## Concepts Demonstrated
* **Object-Oriented Design:** Strong use of encapsulation and logical class hierarchy.
* **GUI Development:** Creation of modern-looking custom Swing/AWT components and their integration into a `JFrame`.
* **Collection Framework:** Managing data within the Java Collections ecosystem.
* **System Integration:** Linking separate logical layers (data, logic, view) in a structured way.
